h1. Maven archetypes

fast project bootstrap with some maven archetypes

h2. installing

```
$ git clone git@github.com:dbourdette/maven-archetypes.git
$ cd maven-archetypes
$ mvn install
```

h2. usage

```
$ mvn archetype:generate
    -DarchetypeGroupId=com.github.dbourdette.maven-archetypes
    -DarchetypeArtifactId=xxx
    -DarchetypeVersion=1.0.0-SNAPSHOT
```

h2. testing generated project

simply run ```mvn tomcat:run```
